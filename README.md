# Quran With Tahir

An online Quran learning platform connecting students with qualified tutors for
personalized Quran education.

🌐 **Live**:
[warm-web-core-main.vercel.app](https://warm-web-core-main.vercel.app)

---

## ⚠️ Important Rules

- **DO NOT** push directly to `main` branch
- **DO NOT** merge anything without project owner's approval
- All changes must go through **Pull Requests** with preview URLs
- Read [CONTRIBUTING.md](./CONTRIBUTING.md) before making any changes

## Quick Start

```bash
# 1. Clone the repo (or your fork)
git clone https://github.com/ilyastahir2001/quran-with-tahir.git
cd quran-with-tahir

# 2. Install dependencies
npm install

# 3. Set up environment variables
cp .env.example .env
# Fill in your values (ask project owner for keys)

# 4. Start dev server
npm run dev
```

Visit `http://localhost:5173`
